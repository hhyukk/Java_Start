# if문과 else
조건이 참일 때만 특정 코드를 실행하고 싶을 때 사용합니다.

```java
if (age >= 18) {
    System.out.println("성인입니다.");
} else {
    System.out.println("미성년자입니다.");
}
```
- if만 사용하면 조건이 참일 때만 실행

- else는 조건이 거짓일 때 실행

# `else if` 문
여러 조건을 순차적으로 검사할 때 사용합니다.
```java
if (age <= 7) {
    System.out.println("미취학");
} else if (age <= 13) {
    System.out.println("초등학생");
} else if (age <= 16) {
    System.out.println("중학생");
} else if (age <= 19) {
    System.out.println("고등학생");
} else {
    System.out.println("성인");
}
```
위에서부터 차례대로 검사하며, **첫 번째로 조건이 참인 곳만 실행**

else는 모든 조건이 거짓일 때 실행

# if vs else if
```java
if (조건1) {
    // 실행
}
if (조건2) {
    // 실행
}
```
- 두 조건이 **독립적**일 때는 `if`를 **각각 따로** 사용해야 합니다.

- 할인 조건처럼 **중복 적용**이 필요할 때는 `else if`를 쓰면 안 됩니다.

#### if문 중괄호 {} 생략 가능할까?
자바에서는 if, else if, else에 **실행할 코드가 한 줄**일 경우 중괄호 {}를 생략할 수 있습니다.
```java
if (age >= 18)
    System.out.println("성인입니다.");
```

#### 두 줄 이상이면 반드시 중괄호 사용해야 함
```java
if (age >= 18)
    System.out.println("성인입니다.");
    System.out.println("환영합니다!"); // <-- 이 줄은 if문과 무관하게 항상 실행됨!
```

#### 중괄호는 언제 쓰는 게 좋을까?
비록 자바 문법상 생략 가능하더라도, 일반적으로는 **중괄호를 항상 사용하는 것이 좋습니다.**

- **가독성 향상**
    - 조건문 범위를 명확하게 보여주어, 코드 읽기 쉬움

- **유지보수에 유리**
    - 나중에 코드 추가할 때 실수 방지

    - 실수로 if문 외부에 코드가 작성되는 문제 예방

> **"한 줄이라도 중괄호를 쓰자!"**
실수 줄이고 협업 시 가독성도 좋아지는 안전한 습관입니다.

# `switch`문
**특정 값에 따라 다르게 동작할 때** switch문이 더 간결할 수 있습니다.
```java
int grade = 2;
switch (grade) {
    case 1: System.out.println("쿠폰 1000원"); break;
    case 2: System.out.println("쿠폰 2000원"); break;
    case 3: System.out.println("쿠폰 3000원"); break;
    default: System.out.println("기본 쿠폰 500원");
}
```
- 값이 일치하는 case 블록을 실행

- break를 꼭 써야 **다른 case로 넘어가지 않음**

## 자바 14 이후 문법 (표현식 형태)
```java
int coupon = switch (grade) {
    case 1 -> 1000;
    case 2, 3 -> 3000;
    default -> 500;
};
```

# 삼항 연산자
`if-else`를 짧게 쓰고 싶을 때 유용합니다.
```java
String status = (age >= 18) ? "성인" : "미성년자";
```
#### 형식
```
(조건) ? 참일 때 값 : 거짓일 때 값;
```

# 문제와 풀이
## 학점 계산기
학생의 점수를 기반으로 학점을 출력하는 자바 프로그램을 작성하자. 다음과 같은 기준을 따른다.

- 90점 이상: "A"
- 80점 이상 90점 미만: "B"
- 70점 이상 80점 미만: "C"
- 60점 이상 70점 미만: "D"
- 60점 미만: "F"

점수는 변수( `int score` )로 지정하고, 해당 변수를 기반으로 학점을 출력하자.

```java
public class ScoreEx {
    public static void main(String[] args) {
        int grade = 92;

        String score = switch (grade / 10) {
            case 10, 9 -> "A";
            case 8 -> "B";
            case 7 -> "C";
            case 6 -> "D";
            default -> "F";
        };
        System.out.println("score = " + score);
    }
}
```

## 거리별 이동 수단
주어진 거리에 따라 가장 적합한 운송 수단을 선택하는 프로그램을 작성하자. 다음과 같은 기준을 따른다.

- 거리가 1km 이하이면: "도보"
- 거리가 10km 이하이면: "자전거"
- 거리가 100km 이하이면: "자동차"
- 거리가 100km 초과이면: "비행기"

거리는 변수( `int distance` )로 지정하고, 해당 변수를 기반으로 운송 수단을 출력하자.

```java
public class DistanceEx {
    public static void main(String[] args) {
        int distance = 6;

        if (distance <= 1) {
            System.out.println("도보를 이용하세요.");
        } else if (distance <= 10) {
            System.out.println("자전거를 이용하세요.");
        } else if (distance <= 100) {
            System.out.println("자동차를 이용하세요.");
        } else {
            System.out.println("비행기를 이용하세요.");
        }
    }
}
```

## 환율 계산
특정 금액을 미국 달러에서 한국 원으로 변환하는 프로그램을 작성하자. 환율은 1달러당 1300원이라고 가정하자. 다음과 같은 기준을 따른다.

- 달러가 0미만이면: "잘못된 금액입니다."
- 달러가 0일 때: "환전할 금액이 없습니다."
- 달러가 0 초과일 때: "환전 금액은 (계산된 원화 금액)원입니다."

금액은 변수( `int dollar` )로 지정하고, 해당 변수를 기반으로 한국 원으로의 환전 금액을 출력하자.

```java
public class ExchangeRateEx {
    public static void main(String[] args) {
        int dollar = 10;

        if (dollar < 0) {
            System.out.println("잘못된 금액입니다.");
        } else if (dollar == 0) {
            System.out.println("환전할 금액이 없습니다.");
        } else {
            System.out.println("환전 금액은 " + dollar * 1300 + "원 입니다.");
        }
    }
}
```

## 영화 추천
요청한 평점 이상의 영화를 찾아서 추천하는 프로그램을 작성하자.

- 어바웃타임 - 평점9
- 토이 스토리 - 평점8
- 고질라 - 평점7

평점 변수는 `double rating` 을 사용하세요. if 문을 활용해서 문제를 풀자.

```java
public class MovieRateEx {
    public static void main(String[] args) {
        double rating = 7;

        if (rating <= 9) {
            System.out.println("'어바웃타임'을 추천합니다.");
        }
        if (rating <= 8) {
            System.out.println("'토이 스토리'를 추천합니다.");
        }
        if (rating <= 7) {
            System.out.println("'고질라'를 추천합니다.");
        }
    }
}
```

## 성취도 출력
String grade 라는 문자열을 만들고, 학점에 따라 성취도를 출력하는 프로그램을 작성하자. 각 학점은 다음과 같은 성취도를 나타낸다.

- "A": "탁월한 성과입니다!"
- "B": "좋은 성과입니다!"
- "C": "준수한 성과입니다!"
- "D": "향상이 필요합니다."
- "F": "불합격입니다."
- 나머지: "잘못된 학점입니다."

`switch` 문을 사용해서 문제를 해결하자.

```java
public class GradeSwitchEx {
    public static void main(String[] args) {
        String grade = "F";

        switch (grade) {
            case "A":
                System.out.println("탁월한 성과입니다!");
                break;
            case "B":
                System.out.println("좋은 성과입니다!");
                break;
            case "C":
                System.out.println("준수한 성과입니다!");
                break;
            case "D":
                System.out.println("향상이 필요합니다.");
                break;
            case "F":
                System.out.println("불합격입니다.");
                break;
            default:
                System.out.println("잘못된 학점입니다.");
        }
    }
}
```

## 더 큰 숫자 출력
두 개의 정수 변수 a 와 b 를 가지고 있다. a 의 값은 10 이고, b 의 값은 20 이다. 삼항 연산자를 사용하여 두 숫자 중 더 큰 숫자를 출력하는 코드를 작성하자.

```java
public class CondOpEx {
    public static void main(String[] args) {
        int a = 10;
        int b = 20;

        int max = a > b ? a : b;

        System.out.println("max = " + max);
    }
}
```
